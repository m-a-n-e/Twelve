# Twelve - O Robô em Three.js

Este projeto apresenta o robô "Twelve" renderizado com Three.js e Next.js. O modelo 3D foi criado por TwistedY e está disponível no Sketchfab: **Twelve - The Robot (Low Poly)**.

## Recursos
- Modelo 3D com **41 partes móveis**
- **Animações dinâmicas** para interatividade
- Renderização otimizada em Three.js

## Como interagir
Clique nos botões **1, 2, 3 ou 4** para escolher uma animação do robô Twelve e veja seus movimentos em ação!

## Tecnologias utilizadas
- **Three.js** para renderização 3D
- **Next.js** para estruturação do projeto
